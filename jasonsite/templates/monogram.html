Monogram

The monogram project was at first a difficult project. It was the first time that I had worked with Illustrator, never mind the fact that I had barely a month of messing around with the Adoboes Creative Cloud Suite of programs. Couple that with using the path tool to create the monogram. I felt much frustrations, but as you can see to your left, I came out ahead. My base monogram image is the "finished project". But, that wasn't the only part of the project. We were tasked on creating a set of stationary objects that would use our logo. So I ended up changing my color and stroke lines. As seen below.
